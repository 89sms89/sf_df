### Какой кейс решаем?
Нужно написать программу, которая угадывает число за минимальное количество попыток.

**Условия соревнования:**
- Компьютер зададывает целое число от 1 до 100, и нам нужно его угадать. Под "угадать" подразумевается "написать программу, которая угадывает число".
- Алгоритм учитывает информацию о том, больше ли случайное число или меньше нужного нам.

**Метрика качества**  
Результаты оцениваются по среднему количеству попыток при 1000 повторений.

**Что практикуем**  
- Учимся писать хороший код на Python;
- Учимся работать с IDE;
- Учимся работать с GitHub.


### Краткая информация о данных
Для поиска загаданного числа введен диапазон чисел от 1 до 101.
